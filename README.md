# Buoyancy Balance Game (Hong Kong Primary)

This repository hosts an interactive buoyancy game for primary students.

## Live Website (GitHub Pages)
After enabling Pages, your site URL will be:

`https://<your-github-username>.github.io/<your-repo-name>/`

## Files
- `index.html` - Main game page
- `reference.png` - Learning reference image
- `buoyancy-game-hk.html` - Working source copy (same game)

## Publish to GitHub (Quick Steps)
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. In GitHub repo: `Settings` -> `Pages`.
4. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main` and folder `/ (root)`
5. Save and wait 1-2 minutes.
6. Open the Pages URL.

## Optional Command Line Setup
If you have Git installed locally:

```bash
git init
git add .
git commit -m "Initial buoyancy game website"
git branch -M main
git remote add origin https://github.com/<your-github-username>/<your-repo-name>.git
git push -u origin main
```
